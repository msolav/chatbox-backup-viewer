# Chatbox Backup Viewer
An HTML page to view and export your interactions with LLMs chatboxes such as Chat-GPT, Claude, etc.

Step 1. Export your conversation(s) from either Chat-GPT or Claude (more services will be included in the future).
Step 2. Wait until you obtain the zipped files, then unzip.
Step 3. Download and open the Chatbox Backup Viewer HTML file in a browser (tested on Google Chrome only).
Step 4. Either drop or load via dialog the unzipped folder or a folder containing all your folders. Alternatively, you may selectively drop JSON files.
Step 5. Explore or export the content of your conversations.

That's it for now. I made this tool because I wanted to delete my Chat-GPT accounts but retain access to my messages, and the provided JSON files were not all that helpful on their own.
